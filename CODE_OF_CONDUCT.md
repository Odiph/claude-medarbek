# Code of Conduct

This project follows a simple, friendly rule: **be kind and be respectful.**

- Treat everyone with respect. No harassment, discrimination, or personal
  attacks.
- Keep contributed verbs playful, not offensive — nothing targeting a person,
  group, religion, or politics.
- Assume good faith. Disagree about the code, not the people.

Maintainers may edit, remove, or reject contributions and comments that violate
these principles.

To report a problem, open an issue or contact the maintainer via the repository.
This is a lightweight adaptation of the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org).
